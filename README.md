# CodePen Pro - Веб-версия

Мощный онлайн редактор кода с поддержкой HTML, CSS и JavaScript.

## 🌐 Демо

**Попробуйте прямо сейчас:** [https://maybeyoou.ru/codepen](https://maybeyoou.ru/codepen)

## 📦 Другие версии

- **🖥️ Electron приложение**: [ветка `electron`](https://github.com/Maybeoff/codepen/tree/electron)
- **🚀 Серверная часть**: [ветка `server`](https://github.com/Maybeoff/codepen/tree/server)

## ✨ Возможности

### 🎨 Редактор кода
- **Подсветка синтаксиса** для HTML, CSS и JavaScript
- **Автодополнение** и подсказки
- **Сворачивание блоков кода** (code folding)
- **Множественные темы** редактора (Dracula, Monokai, Material, Solarized)
- **Форматирование кода** с помощью Prettier
- **Поиск и замена** с регулярными выражениями

### 🖥️ Превью и отладка
- **Живое превью** с автообновлением
- **Адаптивные размеры** (Desktop, Tablet, Mobile)
- **Консоль JavaScript** с отображением ошибок
- **Полноэкранный режим** превью
- **Игнорирование alert/confirm/prompt** для удобства разработки

### 🎭 Темы интерфейса
- **4 глобальные темы**: Светлая, Тёмная, Синяя, Фиолетовая
- **Инъекция CSS темы** в проекты
- **Сохранение настроек** в localStorage

### 📚 Библиотеки
Быстрое подключение популярных библиотек:
- jQuery
- React
- Vue 3
- Tailwind CSS
- Bootstrap CSS
- Three.js

### 💾 Управление проектами
- **Множественные проекты** с переключением
- **Автосохранение** каждую секунду
- **Экспорт в ZIP** с готовой структурой файлов
- **Импорт из ZIP** архивов
- **Поделиться ссылкой** на проект (сжатие через LZ-String)
- **Поделиться с полноэкранным режимом** (добавьте `?fullscreen` к любой ссылке)

> ⚠️ **Примечание**: Функции коротких ссылок, Raw режима и QR-кодов временно недоступны из-за проблем с хостингом API

### ❄️ Дополнительные эффекты
- **Эффект снегопада** с возможностью включения/выключения
- **Разные типы снежинок** с анимацией падения и покачивания

### 📱 PWA поддержка
- **Service Worker** для офлайн работы
- **Манифест** для установки как приложение
- **Кэширование ресурсов** для быстрой загрузки

## 🚀 Технологии

- **Vanilla JavaScript** - основная логика
- **CodeMirror 5.65.2** - редактор кода
- **Prettier** - форматирование кода
- **JSZip** - работа с архивами
- **LZ-String** - сжатие данных для ссылок
- **Service Worker** - PWA функциональность

📦 **Все библиотеки загружаются локально** - подробности в [LIBRARIES.md](LIBRARIES.md)

## 🎯 Использование

1. Откройте [https://maybeyoou.ru/codepen](https://maybeyoou.ru/codepen)
2. Выберите вкладку (HTML, CSS или JS)
3. Начните писать код
4. Наблюдайте результат в превью
5. Используйте настройки для кастомизации
6. Сохраняйте и делитесь проектами

## ⚙️ Горячие клавиши

- `Ctrl + Enter` - Запустить код
- `Ctrl + S` - Сохранить проект
- `Ctrl + /` - Комментировать/раскомментировать
- `Ctrl + Q` - Свернуть/развернуть блок кода
- `Ctrl + F` - Поиск
- `Ctrl + H` - Замена
- `Ctrl + G` - Перейти к строке
- `F11` - Полноэкранный режим превью
- `Escape` - Закрыть модальные окна
